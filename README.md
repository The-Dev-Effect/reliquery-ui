# Reliquery UI
Frontend for Reliquery
Science's Artifact Antiformat

### Local Install
```bash
cd reliquery-ui
pip install -e .
cd frontend
yarn install
```
### Start Services
new shell
```bash
cd frontend
yarn build
uvicorn reliquery_ui.main:app --reload
```
for dev gui
```bash
cd frontend
yarn start
```

### Link Location
A Relic can be shared via a link like:
<br />
{server}/reliquery/relic_type/relic_name
<br />
http://localhost:8000/reliquery/basic/tutorial

### Web Api Exploration
{server}/redoc
<br />
http://localhost:8000/redoc
